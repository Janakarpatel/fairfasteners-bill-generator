# Fair Fasteners — Bill Generator

Monorepo root for the **Fair Fasteners invoice (bill) generator** web app.

## App location

The Next.js application lives in **`fair-bill-app/`**. See that folder’s [`README.md`](fair-bill-app/README.md) and [`docs/ARCHITECTURE.md`](fair-bill-app/docs/ARCHITECTURE.md) for setup, features, and architecture.

## Quick start

```bash
cd fair-bill-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
